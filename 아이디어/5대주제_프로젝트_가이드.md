# 5대 주제 프로젝트 가이드

> **핵심 철학**: 한 권 한 권은 다양한 렌즈로 보되,  
> **실제 프로젝트는 여러 권을 융합**하여 깊이 있게 만든다!

---

## 🎯 독서 → 프로젝트 3단계 구조

```mermaid
graph TD
    A[1단계: 개별 독서] --> B[2단계: 융합 독서]
    B --> C[3단계: 통합 프로젝트]
    
    A --> A1[책 1권을<br/>10가지 렌즈로]
    A1 --> A2[관점 확장<br/>통찰 발견]
    
    B --> B1[같은 주제<br/>여러 권 연결]
    B1 --> B2[패턴 인식<br/>통합 이해]
    
    C --> C1[5대 주제별<br/>실전 프로젝트]
    C1 --> C2[여러 분야 융합<br/>실제 문제 해결]
    
    style A fill:#e3f2fd,color:#111
    style B fill:#fff9c4,color:#111
    style C fill:#c8e6c9,color:#111
```

## 💗 주제 1: 감정 (Emotion)

### 📚 독서 클러스터 (학년별 추천)

```mermaid
mindmap
  root((감정 주제<br/>도서 클러스터))
    초등
      100만번 산 고양이
      괴물들이 사는 나라
      마당을 나온 암탉
    중등
      데미안
      아몬드
      언어의 온도
    고등
      참을 수 없는 존재의 가벼움
      죄와 벌
      예술의 위로
    대학
      훅
      설득의 심리학
      사용자 경험 디자인
```

### 1단계: 개별 독서 (각 책을 10가지 렌즈로)

**예시: 『아몬드』 분석**

| 렌즈 | 질문 | 통찰 |
|------|------|------|
| 💰 경제 | 감정 장애가 경제활동에? | 공감 능력 = 협업 능력 = 경제 가치 |
| ⚖️ 윤리 | 다름을 차별하는 것은? | 신경다양성 존중 필요 |
| 🧠 심리 | 감정은 학습 가능한가? | 감정도 훈련으로 발달 |
| 🏛️ 정치 | 소수자 권리는? | 포용적 정책 필요 |
| 🎨 문화 | 한국 사회 공감 문화는? | 집단주의 vs 개인 차이 |

→ **개인 통찰**: "감정 능력도 근육처럼 훈련할 수 있다"

---

### 2단계: 융합 독서 (여러 권 연결)

**융합 독서 리스트: 감정 이해와 공감**

| 책 | 핵심 통찰 | 융합 질문 |
|-----|----------|-----------|
| 아몬드 | 감정 장애, 다름 이해 | 어떻게 다름을 받아들일까? |
| 언어의 온도 | 말의 힘, 공감 소통 | 어떻게 따뜻하게 표현할까? |
| 설득의 심리학 | 감정 트리거, 설득 | 어떻게 마음을 움직일까? |
| 훅 | 감정과 습관 형성 | 어떻게 습관을 만들까? |

**통합 질문:**
1. 감정은 어떻게 작동하는가? (메커니즘)
2. 공감 능력을 어떻게 키울 수 있나? (훈련)
3. 감정을 어떻게 디자인할 수 있나? (적용)

**융합 통찰:**
> "감정은 ① 이해할 수 있고 (아몬드)  
> ② 표현할 수 있으며 (언어의 온도)  
> ③ 설득에 활용하고 (설득의 심리학)  
> ④ 제품에 적용할 수 있다 (훅)"

---

### 3단계: 통합 프로젝트 (실전)

#### 프로젝트 A: 공감 능력 향상 앱 (중·고등)

**문제 정의:**
- 현대 청소년 공감 능력 저하
- SNS 피상적 소통 증가
- 감정 표현 어려움

**독서 기반 솔루션 설계:**
```mermaid
graph LR
    A[아몬드] -->|감정 훈련 가능| D[앱 기능]
    B[언어의 온도] -->|따뜻한 표현| D
    C[훅] -->|습관 형성| D
    
    D --> E[매일 감정 일기]
    D --> F[공감 대화 연습]
    D --> G[칭찬 미션]
    D --> H[감정 카드 게임]
```

**12주 프로젝트 로드맵:**
```
Week 1-2: 독서 & 사용자 리서치
• 4권 완독 + 10가지 렌즈 분석
• 청소년 30명 인터뷰

Week 3-4: 융합 분석 & 아이디어
• 4권의 통찰 통합
• HMW 질문 100개
• 솔루션 컨셉 3개

Week 5-6: MVP 설계
• 화면 설계 (Figma)
• 핵심 기능 정의
• 습관 형성 메커니즘 설계

Week 7-9: 개발 & 테스트
• 노코드 개발 (Bubble)
• 베타 테스트 50명
• 피드백 수집

Week 10-11: 개선 & 확장
• 주요 이슈 개선
• 추가 기능 구현
• 사용자 100명 확보

Week 12: 발표 & 포트폴리오
• 피칭 덱 제작
• Demo Day 발표
• 포트폴리오 정리
```

**기대 효과:**
- 공감 점수 30% 향상
- 긍정적 소통 2배 증가
- 감정 표현 능력 향상

---

#### 프로젝트 B: 치유 콘텐츠 큐레이션 서비스 (대학)

**문제 정의:**
- 정신건강 문제 증가
- 적절한 콘텐츠 찾기 어려움
- 개인별 맞춤 필요

**독서 기반 솔루션:**
- 감정 상태 진단 (심리학 기반)
- 맞춤형 콘텐츠 추천 (AI 활용)
- 커뮤니티 형성 (관계 이론)

**비즈니스 모델:**
- Freemium (기본 무료, 프리미엄 유료)
- B2B (기업 웰빙 프로그램)
- 콘텐츠 제작자 수익 분배

## 💰 주제 2: 돈 (Money)

### 📚 독서 클러스터 (학년별 추천)

```mermaid
mindmap
  root((돈 주제<br/>도서 클러스터))
    초등
      용돈 관리 동화
      경제 그림책
    중등
      괴짜 경제학
      넛지
      총 없는 전쟁
    고등
      국부론
      블루오션 전략
      21세기 자본
    대학
      린 스타트업
      제로 투 원
      그로스 해킹
      플랫폼 레볼루션
```

### 2단계: 융합 독서

**융합 독서 리스트: 가치 창출의 메커니즘**

| 책 | 핵심 통찰 | 키워드 |
|-----|----------|--------|
| 국부론 | 분업, 시장 원리 | 생산성 |
| 괴짜 경제학 | 숨은 인센티브 | 동기 |
| 넛지 | 선택 설계 | 행동 |
| 린 스타트업 | 빠른 검증 | 실험 |
| 제로 투 원 | 독점 시장 | 혁신 |
| 블루오션 전략 | 새 시장 창조 | 차별화 |
| 그로스 해킹 | 성장 실험 | 확장 |

**통합 질문:**
1. 가치는 어디서 오는가? (국부론, 제로 투 원)
2. 어떻게 시장을 만드는가? (블루오션, 플랫폼)
3. 어떻게 빠르게 성장하는가? (린, 그로스)

**융합 통찰:**
> "가치 창출 = ① 차별화된 제품 (제로 투 원)  
> ② 새로운 시장 (블루오션)  
> ③ 빠른 검증 (린)  
> ④ 성장 메커니즘 (그로스)"

---

### 3단계: 통합 프로젝트

#### 프로젝트 A: 학생 맞춤 수익 창출 (고등·대학)

**문제 정의:**
- 용돈 부족, 경제 독립 욕구
- 아르바이트 vs 학업 갈등
- 창의적 수익 모델 필요

**독서 기반 솔루션 설계:**

**Step 1: 시장 분석 (괴짜 경제학 + 넛지)**
```
내가 가진 것:
• 시간: 방과 후 2-3시간
• 기술: SNS, 디자인, 글쓰기
• 네트워크: 학교 친구 500명

숨은 기회:
• 학생들의 고민 (시험, 진로)
• 학부모의 니즈 (교육 정보)
• 학교의 문제 (비효율)
```

**Step 2: 아이디어 발상 (블루오션 + 제로 투 원)**
```
4가지 액션 프레임워크:
• 제거: 높은 고정비
• 감소: 시간 투입
• 증가: 자동화
• 창조: 새로운 가치

3가지 비즈니스 모델:
1. 정보 플랫폼 (시험 정보 공유)
2. 재능 마켓 (디자인, 과외)
3. 커뮤니티 구독 (프리미엄 콘텐츠)
```

**Step 3: MVP 제작 (린 스타트업)**
```
Week 1-2: 고객 인터뷰 30명
Week 3-4: 랜딩 페이지 + 결제
Week 5-6: 베타 런칭 (친구 50명)
Week 7-8: 피벗 or 스케일
```

**Step 4: 성장 실험 (그로스 해킹)**
```
AARRR 지표:
• Acquisition: SNS 바이럴
• Activation: 첫 구매 유도
• Retention: 매주 가치 제공
• Revenue: 프리미엄 전환
• Referral: 추천 인센티브
```

**실제 사례 목표:**
- 3개월: 월 50만원 수익
- 6개월: 월 200만원 수익
- 12개월: 학비 자립

---

#### 프로젝트 B: 지역 소상공인 컨설팅 (대학)

**독서 융합:**
- 경쟁 전략: 산업 분석
- 블루오션: 차별화 전략
- 넛지: 매장 동선 설계
- 그로스: 재방문 전략

**12주 프로젝트:**
1. 주변 소상공인 5곳 선정
2. 7권의 책으로 진단 프레임워크
3. 맞춤 전략 제안
4. 3개월 컨설팅 실행
5. 매출 증대 측정

**포트폴리오:**
- 전략 보고서
- 실제 매출 증대 데이터
- 추천서

---

### 3단계: 통합 프로젝트

#### 프로젝트 A: 동아리/학생회 혁신 (중·고등)

**문제:**
- 형식적 활동
- 참여율 저조
- 갈등 관리 어려움

**독서 기반 솔루션:**

**1. 조직 문화 설계 (실리콘밸리의 팀장들)**
```
Radical Candor 적용:
• 1:1 미팅 (격주)
• 직접적 피드백 문화
• 칭찬 공개, 비판 개인
```

**2. 권한 위임 (멀티플라이어)**
```
5가지 실천:
• 역할 명확화
• 의사결정권 위임
• 실수 허용 문화
• 성장 기회 제공
• 성과 인정
```

**3. 소통 시스템 (언어의 온도)**
```
따뜻한 소통:
• 감사 메시지 (주간)
• 고민 나눔 (월간)
• 축하 이벤트 (수시)
```

**12주 실험:**
- 참여율 50% → 80% 증가
- 만족도 3점 → 4.5점
- 프로젝트 완수율 2배

#### 프로젝트 B: 커뮤니티 빌딩 플랫폼 (대학)

**MVP:**

## 🧘 주제 4: 성찰 (Reflection)

### 📚 독서 클러스터

```mermaid
mindmap
  root((성찰 주제<br/>도서 클러스터))
    초등
      아홉 살 인생
      생각하는 ㄱㄴㄷ
    중등
      데미안
      소피의 세계
      연금술사
    고등
      이방인
      참을 수 없는 존재
      존재와 시간
    대학
      7가지 습관
      아토믹 해빗
      Deep Work
```

### 2단계: 융합 독서

**융합 주제: 자기 이해와 성장 시스템**

| 책 | 핵심 통찰 | 실천 |
|-----|----------|------|
| 데미안 | 자기 되기 | 정체성 |
| 이방인 | 진정성 | 가치관 |
| 7가지 습관 | 원칙 중심 | 시스템 |
| 아토믹 해빗 | 1% 개선 | 복리 효과 |
| Deep Work | 몰입 | 생산성 |

**통합 질문:**
1. 나는 누구인가? (정체성)
2. 무엇을 원하는가? (가치관)
3. 어떻게 성장하는가? (시스템)

---

### 3단계: 통합 프로젝트

#### 프로젝트 A: 개인 성장 시스템 구축 (고등·대학)

**독서 기반 설계:**

**1. 자기 이해 (데미안 + 이방인)**
```
8주 자기 탐구:
• Week 1-2: 인생 타임라인
• Week 3-4: 가치관 워크숍
• Week 5-6: 강점 발견
• Week 7-8: 사명선언문
```

**2. 목표 설정 (7가지 습관)**
```
OKR 프레임워크:
• Objective: 이루고 싶은 것
• Key Results: 측정 가능한 결과
• 분기별 리뷰
```

**3. 습관 설계 (아토믹 해빗)**
```
습관 스택:
• 아침: 명상 5분 → 독서 30분 → 운동 20분
• 저녁: 일기 10분 → 계획 5분
• 주간: 리뷰 1시간
```

**4. 몰입 환경 (Deep Work)**
```
생산성 시스템:
• 시간 블록킹
• 디지털 디톡스
• 몰입 공간 설계
```
